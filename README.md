# iptables-docker-enhanced

基于 iptables-docker 的增强版本，支持通过配置文件简单管理防火墙规则。

## 快速开始

```bash
# 安装
sudo bash scripts/install.sh

# 编辑配置
sudo iptables-docker.sh edit

# 查看状态
sudo iptables-docker.sh status
```
